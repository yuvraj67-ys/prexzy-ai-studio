# Prexzy AI Studio

## Deploy in 3 Steps (No Terminal):

### Step 1: GitHub
1. Go to github.com → New Repository → Name: "prexzy-ai-studio"
2. Upload all files from this project (drag & drop)
3. Click "Commit changes"

### Step 2: Vercel
1. Go to vercel.com → "Add New Project"
2. Import your GitHub repository
3. In "Environment Variables" add:
   - Key: `PREXZY_BASE_URL`
   - Value: `https://your-actual-prexzy-api-url.com` (replace with real URL)
4. Click "Deploy"

### Step 3: Done! 🎉
Your app is live at: `your-project.vercel.app`
